# Kotlin Expense Tracker (Android)

State:
- Minimal Android app implemented with Jetpack Compose and in-memory state.
- Single Activity/screen with month navigation, summaries, add/edit/delete, BRL formatting.

Build:
- From this folder:
  - ./gradlew tasks
  - ./gradlew assembleDebug

Requirements:
- Java 17 for AGP 8.x+
- Android SDK 34
- Repositories: google() and mavenCentral()
