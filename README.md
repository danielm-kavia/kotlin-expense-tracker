# kotlin-expense-tracker

Status: Buildable Android project with a minimal Compose app.
- Single screen implements month navigation, summary totals, add/edit/delete, BRL formatting, and a simple transaction list using in-memory state.
- Jetpack Compose + Material3, single Activity.
- No persistence or external services.

Build:
- (cd kotlin-expense-tracker && ./gradlew tasks)
- (cd kotlin-expense-tracker && ./gradlew assembleDebug)
